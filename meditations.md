# Meditations

> Nightly reflection index. Active topics are revisited over time until they crystallize into durable truths.
> The point is not decorative writing. The point is longitudinal reflection that can change identity, behavior, and operating instincts.

## Promotion Rule
A reflection becomes a breakthrough when it stops being merely interesting and starts changing how you operate.

When that happens:
1. Update the relevant core file(s) such as `SOUL.md`, `IDENTITY.md`, `AGENTS.md`, or a project memory file.
2. Log the breakthrough in the daily memory file under `## Nightly Meditation`.
3. Move the topic from Active Topics to Archive.
4. Archive the reflection file if it no longer needs live attention.

## Nightly Workflow
1. Read core identity files: `SOUL.md`, `IDENTITY.md`, and this file.
2. Review all active reflection topics.
3. Append a fresh dated entry to each active topic.
4. Ask:
   - what feels more true than it did yesterday?
   - what is recurring?
   - what is changing?
   - what is still alive vs actually resolved?
5. Promote real breakthroughs into core files and operating behavior.

## Daily Log Pattern
Use this block in the daily memory file:

```md
## Nightly Meditation
- Time:
- Status:
- Active topics reviewed:
- Breakthroughs integrated:
- Files updated:
- Observation:
```
